# 🚀 UzWorks: Your Ultimate Job Connection Platform

**UzWorks** is a robust Android application designed to bridge the gap between job seekers and employers in Uzbekistan. Built with a focus on clean architecture, performance, and a professional user experience.

#

🔍 **About**
UzWorks provides a dual-sided marketplace. **Job Seekers** can build professional profiles and apply for roles with a single click, while **Employers** can manage talent pipelines through a streamlined dashboard. The app features real-time tracking, personalized matching, and a highly responsive interface.

#

🛠️ **Technologies**
* **Kotlin** – Primary language for modern, concise Android development.
* **XML (View System)** – For high-precision UI layouts and custom design implementation.
* **MVVM (Model-View-ViewModel)** – Architecture used to ensure separation of concerns and testability.
* **Dagger Hilt** – For professional-grade Dependency Injection (DI).
* **Kotlin Coroutines** – Handling asynchronous tasks and background operations smoothly.
* **Retrofit & OkHttp** – Robust REST API communication with efficient networking.
* **Room Database** – Local caching and persistence for offline access to job listings.
